## 🚀 Overview

Chess PGN Analyzer is a lightweight web application that lets you load, view, and navigate chess games in PGN format. It is built with React and Vite, uses `chess.js` for chess logic, and integrates Stockfish for engine-based move analysis directly in the browser.

---

## ✨ Features

- **PGN Loader**  
  Load and parse `.pgn` files with ease.

- **Move Navigation**  
  Step through the game move by move using Previous and Next controls.

- **Engine Analysis**  
  Get real-time move evaluation using Stockfish.

- **Board Controls**  
  Reset the board, flip sides, and switch between dark and light mode.

- **Move List Viewer**  
  View the full game in a structured move list.

- **Utility Actions**  
  Copy the current FEN and export the loaded PGN.

---

## 🛠️ Tech Stack

- **Framework:** React + Vite
- **Language:** JavaScript (ES6+)
- **Chess Logic:** [`chess.js`](https://github.com/jhlywa/chess.js/)
- **Engine:** Stockfish (WebAssembly)
- **Styling:** CSS / inline styling
- **APIs:** File API, Clipboard API
- **Deployment:** Vercel
